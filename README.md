# Q2 — Product Cards Grid

A responsive product card layout using CSS Grid.

## How to view
Open `index.html` directly in the browser or enable GitHub Pages from:
Settings → Pages → Build from main branch.

## Requirements covered
- Semantic HTML
- CSS Grid with auto-fit/minmax
- Featured card spans full width on desktop
- Responsive: stacks on mobile
- Box model + padding + border + hover
